# http-parse

A pure Zig HTTP stack, built from the ground up for performance and safety.

## Goals

- Pure Zig implementation (no external C dependencies).
- Support for HTTP/1.1, HTTP/2, and HTTP/3.
- Focus on modularity and standalone parsers.
- Track the latest Zig developments.

## Components

- HTTP/1 parser (Planned)
- TLS implementation (Planned)
- HTTP/2 parser (Planned)
- HTTP/3 parser (Planned)
